## ⚡ Descripción

Proyecto interactivo desarrollado para resolver problemas de **Física II** mediante una interfaz web moderna.

Permite calcular de forma rápida y visual:

- 🔌 Circuitos eléctricos
- ⚡ Ley de Ohm
- 🧲 Ley de Coulomb
- 🔋 Capacitores
- 📊 Potencia eléctrica
- 🧠 Conductividad de materiales
